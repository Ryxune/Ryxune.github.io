Hexo & GitHub免费搭建自己的博客

发表于2023-04-26|更新于2023-04-26|Hexo
|字数总计:317|阅读时长:1分钟|阅读量:

前置环境准备

  • Node
  • Git

Hexo搭建博客

  1. npm i -g hexo-cli 安装Hexo

  2. hexo -v 检查是否安装成功

  3. 新建GitHub仓库,仓库的格式必须为：<用户名>.github.io

  4. 连接GitHub

    1. 生成ssh公钥,路径为 C:\Users\用户名\.ssh\id_rsa.pub

    2. 将 SSH KEY 配置到 GitHub

      1. 进入github，点击右上角头像 选择settings

      2. 进入设置页后选择 SSH and GPG keys

        注意: 要是有【Key type】的选择项 ，选择默认Authentication Key 即可。

    3. ssh -T git@github.com检查配置是否成功

      如下提示表示成功
      Hi Ryxune! You’ve successfully authenticated, but GitHub does not provide shell access.

  5. hexo init blog(项目名) 初始化Hexo项目, cd blog切换到项目目录.

  6. npm i 安装依赖

  7. hexo s 或hexo serve开启本地服务, 即可在浏览器中访问 http://localhost:4000/

  8. git i hexo-deployer-git安装部署插件

  9. 在_config.yml文件中配置仓库路径和分支,注意查看分支是master还是main

  10. hexo clean 清除缓存(删除之前生成的文件，若未生成过静态文件，可忽略此命令。)

  11. hexo g 再次生成静态文件

  12. hexo d 或hexo deploy部署到GitHub

  13. 用户名.gitee.io访问
